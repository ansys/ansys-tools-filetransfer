from_server_address#

classmethod Client.from_server_address(server_address, *, max_message_length=131072)#

Initialize the client from a server URL.

Parameters:

  • server_address (str) – IPv4/IPv6 address and port of the server to connect to. For example, 10.0.0.42:12345.

  • max_message_length (int, default: 131072) – Maximum message length in bytes to send over the channel.

Return type:

Client

Returns:

Instantiated FileTransfer client.
